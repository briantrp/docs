Follow these steps to get your first Helvi instance running and configured.

<Steps>
  <Step title="Clone & Install">
    Clone the repository and install dependencies using your preferred package manager.
    ```bash
    git clone https://github.com/helvi-hq/helvi
    cd helvi
    npm install
    ```
  </Step>
  <Step title="Environment Setup">
    Copy the `.env.example` to `.env` and configure your database and authentication secrets.
    ```bash
    cp .env.example .env
    ```
    <Info>Ensure `NEXTAUTH_URL` is set correctly for your environment (e.g., `http://localhost:3000`).</Info>
  </Step>
  <Step title="Database Initialization">
    Run the Prisma migrations to set up your database schema.
    ```bash
    npx prisma migrate dev
    ```
  </Step>
  <Step title="Launch the Platform">
    Start the development server.
    ```bash
    npm run dev
    ```
    Navigate to `http://localhost:3000` to see your Helvi instance in action.
  </Step>
</Steps>
